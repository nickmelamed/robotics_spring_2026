# Git Resources

Git is a version control system; essentially, it allows you to track changes in your files. 

Below is the structure of this directory: 

```text 
├── git_resources
│   ├── CONTRIBUTING.md
│   ├── git_primer
│   │   ├── 00-why-version-control.md
│   │   ├── 01-git-mental-model.md
│   │   ├── 02-core-workflow.md
│   │   ├── 03-branching.md
│   │   ├── 04-merge-vs-rebase.md
│   │   ├── 05-resolving-conflicts.md
│   │   ├── 06-solving-common-issues.md
│   │   ├── exercises
│   │   │   ├── exercise-1-basic-workflow.md
│   │   │   ├── exercise-2-branching.md
│   │   │   └── exercise-3-conflicts.md
│   │   └── README.md
│   └── README.md
└── README.md
```

Our `CONTRIBUTING.md` file outlines rules to follow to ensure our repository is orderly, and the `git_primer` folder contains a quick overview of using git and some exercises to verify your knowledge.
